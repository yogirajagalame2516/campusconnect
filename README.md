# CampusConnect

CampusConnect is a Smart Campus Grievance Management System developed using Spring Boot, Thymeleaf, MySQL and JPA.

## Features

* Student Complaint Submission
* Complaint Tracking by Complaint ID
* Admin Login with Session Management
* Complaint Search and Filters
* Priority and Status Management
* Admin Notes
* Complaint PDF Download
* Complaint Details Page
* MySQL Database Integration

## Technologies Used

* Java 17
* Spring Boot
* Spring Data JPA
* Thymeleaf
* MySQL
* Maven
* HTML/CSS
